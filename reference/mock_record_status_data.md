# Mock Record Status Dashboard data

A synthetic dashboard tile table generated from `mock_redcap_project`
with REDCapExploreR's internal record-status logic. This dataset is
suitable for examples of
[`plot_record_status()`](https://chop-cgtinformatics.github.io/REDCapExploreR/reference/plot_record_status.md).

## Usage

``` r
mock_record_status_data
```

## Format

A tibble with one row per record, event, and form tile.

## Source

Generated by `data-raw/mock-data.R`.
